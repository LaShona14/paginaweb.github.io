Definiciones HTML

HTML (Lenguaje de Marcado de Hipertexto) es el lenguaje estándar para crear y diseñar páginas web.

<HEAD>

La etiqueta <HEAD> contiene información de encabezado del documento HTML.

<TITLE>

La etiqueta <TITLE> define el título del documento HTML, visible en la barra de título del navegador.

<BODY>

La etiqueta <BODY> contiene el contenido visible del documento HTML.

<CENTER>

La etiqueta <CENTER> solía usarse para centrar contenido, pero su uso ha quedado obsoleto.

<BR>

La etiqueta <BR> se utiliza para insertar un salto de línea en el contenido.

<H1>

La etiqueta <H1> define un encabezado de nivel 1, el principal de una sección o página.

<HR>

La etiqueta <HR> inserta una línea horizontal, a menudo utilizada para separar secciones de contenido.

<HTML>

La etiqueta <HTML> envuelve todo el contenido del documento HTML.

<P>

La etiqueta <P> se utiliza para definir párrafos en el contenido HTML.
